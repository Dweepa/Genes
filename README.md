# Generating embeddings for drugs

## Running the demo using docker

```
docker-compose pull
docker-compose up

enter localhost:3000 in the browser
```

## Running the demo via code
### Start the server
```
cd demo_server
python server.py
```


### Start the UI
```
cd UI/minorproj
npm install
npm start
```
